# GoodGarden


## Setup

1. Maak een (venv) virtual environment:

    ```bash
    python -m venv venv
    ```

2. Activeer de (venv) virtual environment:

    ```bash
    venv\Scripts\activate
    ```

3. Installeer de dependencies:

    ```bash
    pip install -r requirements.txt
    ```

4. Run de applicatie:

    ```bash
    python run.py
    ```
